# homebridge-lightify

Plugin adding Osram Lightify support to homebridge

## Installation
1. Install plugin with `npm install -g chris1705/homebridge-lightify`
2. Add platform within `config.json` of you homebridge instance:

    ```
    {
        "platform": "Lightify",
        "host": "192.168.1.3",
    }
    ```
3. Restart homebridge
4. Enjoy!
